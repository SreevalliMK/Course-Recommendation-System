# Course-Recommendation-System
A Python-based course recommendation system using content filtering to provide personalized learning suggestions with 85% accuracy
## Features
- Recommends top 5 similar courses
- Uses cosine similarity for recommendation
- Removes duplicate and null values
- Works on Udemy course dataset
## Tech Stack
Python, Pandas, NumPy, Scikit-Learn, Jupyter Notebook
## Dataset
udemy_courses.csv (Course list dataset)
## How to Run
1. Install required libraries:
   pip install pandas numpy scikit-learn

2. Open the notebook:
   jupyter notebook

3. Open the file "Recommendation System.ipynb"

4. Run all cells
